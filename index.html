Tutorial Básico de Python

Python es un lenguaje de programación fácil de aprender y muy poderoso. Es ideal para principiantes, y también es usado en una gran variedad de proyectos profesionales.

1. Introducción: Ejecutar Python

Para comenzar con Python, crear un archivo .py con tu código y ejecutarlo.

Para ejecutarlo, abre una terminal, navega hasta la ubicación del archivo y escribe:

python nombre_del_archivo.py

2. Variables

Las variables en Python no necesitan ser declaradas explícitamente. Puedes asignarles cualquier tipo de dato. 

# Cadenas
nombre = "Juan"
# Números
edad = 25
# Booleanos
es_mayor = True
# Constantes (convención: en mayúsculas)
PI = 3.14159

Para obtener datos de entrada del usuario, puedes usar la función input(). El valor devuelto por input() es siempre una cadena (string), por lo que si esperas un número, tendrás que convertirlo usando int() o float(). 

# Obtener un valor del usuario
nombre = input("¿Cómo te llamas? ")
print("Hola, " + nombre + "!")

# Convertir la entrada a un número
edad = int(input("¿Cuántos años tienes? "))
print("Tienes", edad, "años.")

3. Operaciones Matemáticas

Python permite realizar operaciones básicas como suma, resta, multiplicación y división.

# Operaciones básicas
suma = 5 + 3
resta = 5 - 3
multiplicacion = 5 * 3
division = 6 / 3

4. Manipulación de Cadenas

Las cadenas (strings) pueden ser manipuladas de varias maneras en Python.

saludo = "Hola " + "mundo"
longitud = len(saludo)
reemplazo = saludo.replace("mundo", "Python")
mayusculas = saludo.upper()
minusculas = saludo.lower()

5. Funciones

Las funciones en Python son bloques de código reutilizables que se ejecutan cuando son llamados. Se pueden definir funciones con o sin parámetros y también pueden devolver valores.

5.1 Funciones Sin Parámetros

def saludar():
    print("¡Hola!")

saludar()  # Llamar a la función

5.1 Funciones Con Parámetros

def calcular_area(base, altura):
area = base * altura / 2
return area

# Llamar a la función con dos argumentos
area_triangulo = calcular_area(10, 5)

# Mostrar el área calculada
print(area_triangulo)  # Salida: 25.0

6. Condicionales

Los condicionales en Python se implementan con las palabras clave if, elif, y else. 

edad = 18

if edad >= 18:
    print("Eres mayor de edad.")
else:
    print("Eres menor de edad.")

7. Bucles

El bucle for en Python se usa para iterar sobre una secuencia (como una lista o un rango de números).

# Bucle for
for i in range(5):
    print("Número:", i)

# Bucle while
contador = 0
while contador < 5:
    print("Contador:", contador)
    contador += 1

Listas

Las listas son colecciones de elementos ordenados. Puedes agregar, eliminar, modificar y acceder a los elementos en una lista.

Ejemplos básicos:


numeros = [1, 2, 3, 4, 5]
    
# Agregar un elemento al final de la lista
numeros.append(6)  

# Eliminar un elemento específico de la lista
numeros.remove(2)  

# Obtener la longitud de la lista
longitud = len(numeros)  

# Mostrar la lista actualizada
print(numeros)  # Salida: [1, 3, 4, 5, 6]

# Acceder al primer elemento
print(numeros[0])  # Salida: 10

# Acceder al tercer elemento
print(numeros[2])  # Salida: 30
